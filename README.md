# Event Planner

A lightweight event planning application written in Python.

## Overview

Event Planner is a simple console application that helps organize upcoming events, meetings, and personal activities.

The project demonstrates working with Python data structures, functions, sorting, and formatted console output.

## Features

- Create a list of events
- Display upcoming activities
- Sort events by date
- Count total planned events
- Clean console interface
- Easy to extend
